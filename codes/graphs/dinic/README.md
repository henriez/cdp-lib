# Dinic - Flow/matchings
\begin{itemize}
    \item \textbf{General Network:} $\mathcal{O}(VE \log U)$.
    \item \textbf{Unit Capacity Network:} $\mathcal{O}(\min(V^{2/3}, E^{1/2}) \cdot E)$. Often considered $\mathcal{O}(E\sqrt{V})$.
    \item \textbf{Bipartite Matching:} $\mathcal{O}(\min(V^{2/3}, E^{1/2}) \cdot E)$. Often considered $\mathcal{O}(E\sqrt{V})$.
\end{itemize}
